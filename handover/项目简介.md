# BibiliDown — B 站视频下载器

## 这是什么

一个**开源免费**的 B 站视频下载工具，支持浏览器直接使用 + Electron 桌面版。粘贴 BV 号即可解析下载，最高支持 8K 画质。

## 核心功能

- **一键解析下载** — 支持 BV / AV / EP / SS / 收藏夹 / 短链接多种格式
- **最高 8K** — DASH 双流 + FLV 单流，360P ~ 8K 全覆盖
- **三种登录** — QR 码扫码 / 密码 / 短信，登录后可下高清 + 大会员内容
- **批量下载** — 多 P 视频、收藏夹一键全下，并发数可调
- **暗色 + 5 色主题** — 亮/暗双模式，蓝紫绿橙红 5 种强调色
- **文件名模板** — 自定义命名规则（标题/BV号/画质/分P号等变量）
- **预设系统** — 保存多套下载配置，一键切换
- **历史记录** — 自动记录，支持日期筛选 + 搜索 + 一键重下

## 技术栈

Electron 43 + React 18 + TypeScript 5 + Vite 5 + Tailwind 3 + Zustand 5 + Phosphor Icons

## 快速开始

```bash
git clone https://github.com/Luo-Feng6/bibilidown
cd bibilidown
npm install
npm run dev              # 浏览器模式 → localhost:5173
npm run dev:electron     # 桌面模式
```
