## 四、核心问题与解决方案

### 4.1 格式选择为什么要拆成视频/音频两个？

**问题链**：
```
用户问"视频可以选择下载格式，音频也是"
↓
当前只有一个 preferredFormat 同时控制视频和音频
↓
用户想视频下 MP4、音频下 MP3 时无法实现
↓
且 App.tsx 里 format: 'mp4' 硬编码，根本没读设置
```

**解决方案**：
1. `userPrefsStore` 新增 `preferredAudioFormat: 'm4a' | 'm4s' | 'mp3'`
2. 设置页拆两行：视频输出格式（MP4/M4S）、音频输出格式（M4A/MP3/M4S）
3. App.tsx 新增 `resolveFormatLabel(mode)` → 根据 downloadMode 组合视频/音频格式标签
4. `download-manager.ts` 浏览器+Electron 双路径读 `preferredAudioFormat` 决定扩展名

**关于 MP3**：B站实际只提供 AAC 编码的 M4A/M4S 容器。浏览器模式下选 MP3 只是改扩展名（`.mp3`），内容仍是 AAC，可能部分播放器不兼容。需要 Electron + FFmpeg 才能真正转码为 MP3。

### 4.2 原生 select 为什么难看？

**问题**：`appearance: none` 只能隐藏浏览器默认箭头，但 `<option>` 下拉弹出部分是操作系统原生渲染，CSS 无法控制——字体、间距、背景色、hover 态全部是 OS 风格，在暗色主题中尤其突兀（Windows 弹出白底黑字）。

**解决方案**：完全放弃 `<select>`，改用 button + 绝对定位 div 模拟：
```
[trigger button] ← 点击切换 open/close
    ↓
[dropdown menu] ← position: absolute, top: 100%
    ├── option 1  (hover → surface-overlay)
    ├── option 2  (selected → accent-muted + ●)
    └── option 3
```
- `useRef` + `useEffect` + `mousedown` listener 实现 click-outside 关闭
- CaretDown 图标 `transform: rotate(180deg)` 动画
- 选中项左边 ● 圆点标记

### 4.3 下载队列已完成项为什么没有 X？

**问题**：第十四轮的 DownloadItem 在 `isDone` 状态下只渲染了一个条件按钮：
```tsx
{isDone && onOpenFolder && (
  <IconButton label="打开文件夹" onClick={...}>
    <FolderOpen />
  </IconButton>
)}
```
浏览器模式不传 `onOpenFolder`（安全限制），所以已完成项右边**完全空白**——用户找不到删除入口。

**解决方案**：改为始终渲染删除 X，文件夹按钮可选：
```tsx
{isDone && (
  <>
    {onOpenFolder && <IconButton ...><FolderOpen /></IconButton>}
    <IconButton label="删除" onClick={onCancel} destructive><X /></IconButton>
  </>
)}
```

### 4.4 附带行为什么会和按钮重叠？

**问题**：内联模式下下载方式行使用的是 `flex flex-wrap`：
```html
<div style="flex-wrap">
  下载方式: [仅视频] [仅音频] [分别下载] [合并] ⚙更多设置→
  <div>附带: [💬弹幕] [📝字幕]</div>  <!-- 这个 div 没有 width 控制 -->
</div>
```
当卡片宽度不够时，附带行 div 和 `⚙ 更多设置 →`（`marginLeft: auto`）挤在同一行，产生重叠。

**解决方案**：附带行加 `flexBasis: '100%'`，强制在 flex 容器中独占一行。

### 关键数据流

```
用户解析 BV → VideoCard 显示
  ├── 清晰度选择（纯画质）
  ├── 内联模式:
  │   ├── 下载方式: [仅视频] [仅音频] [分别下载] [合并]
  │   └── 附带: [💬弹幕 ⬇] [📝字幕 ⬇]
  │        ├── 左边点击 → 切换随视频自动下载
  │        └── 右边 ⬇ → 独立下载（fetchSubtitle/fetchDanmaku → Blob → <a>）
  │
  └── resolveFormatLabel(downloadMode)
      ├── video-only → 'MP4' 或 'M4S'（读 preferredFormat）
      ├── audio-only → 'M4A' 或 'MP3' 或 'M4S'（读 preferredAudioFormat）
      ├── separate/merge → 'MP4+M4A'
      └── undefined → 'MP4'

下载引擎读设置:
  ├── 视频扩展名 → preferredFormat → .mp4 / .m4s
  └── 音频扩展名 → preferredAudioFormat → .m4a / .mp3 / .m4s
```
